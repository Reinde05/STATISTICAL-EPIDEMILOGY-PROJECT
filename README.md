# 📊 Statistical Epidemiology Project

## Overview

This repository contains projects completed for the **Statistical Epidemiology** course. It includes R scripts for data analysis, research papers documenting the methodology and findings, and HTML reports generated from R Markdown and published on RPubs.

## Repository Contents

- 📂 **code/** – R scripts (`.R`) and R Markdown (`.Rmd`) used for data analysis.
- 📂 **paper/** – Research papers and project reports in PDF format.
- 📂 **html/** – HTML reports generated from R Markdown and published on RPubs.
- 📂 **images/** *(optional)* – Figures, plots, and visualizations produced during the analysis.

## Software & Tools

- R
- RStudio
- R Markdown
- RPubs

## Purpose

The purpose of this repository is to document coursework and demonstrate the application of statistical methods in epidemiological research, including data analysis, interpretation, and scientific reporting.
